# Android client instructions

افتح هذا المجلد في Android Studio. المشروع هنا هو هيكل مبسط لبدء التطوير.

الملفات الأساسية:
- app/src/main/java/com/example/vc/MainActivity.kt  -> كود التقاط الصوت، الإرسال للسيرفر، وتشغيل النتيجة.
- app/build.gradle -> إعدادات Gradle الأساسية

ملحوظة: هذا ليس مشروع Android كامل مُجمّع، لكنه skeleton يمكنك فتحه وملؤه بالمزيد من التبعيات وملفات الموارد.
